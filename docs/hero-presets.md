# AEGIS Hero Presets v1

This guide explains how to create reusable hero presets and embed them on pages.

## Create a Hero preset

1. Go to **AEGIS Hero â†’ Heroes** in the WordPress admin.
2. Click **Add New**.
3. Build the hero using the **Aegis Hero** block (the editor is locked to this block).
4. Publish the preset.

## Embed a preset on a page

1. Edit any page or post.
2. Insert the **Aegis Hero Embed** block.
3. Select the preset from the dropdown.
4. Save or publish.

## Notes

- The **Aegis Hero** block is hidden from the regular inserter and is meant for preset editing only.
- The **Aegis Hero Embed** block renders the first **Aegis Hero** block from the selected preset.
