Black/white minimalist branding for the WordPress login screen with configurable logo and visibility toggles.

== Description ==
This plugin provides a minimal black/gray/white theme for the WordPress login screen, a configurable logo via the media library, and options to hide optional login elements.

== Installation ==
1. Upload the plugin folder to `wp-content/plugins/`.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to Settings â†’ Login Branding (B/W) to configure the logo and visibility toggles.
